# Bank Loan Data Analysis Project

## Project Overview

This project performs data analysis on bank loan data to understand customer financial patterns and loan repayment behavior.
The analysis helps identify trends related to income, credit score, employment status, and loan repayment.

## Technologies Used

* Python
* Pandas
* Matplotlib

## Dataset Information

The dataset contains the following features:

* Age
* Annual Income
* Credit Score
* Debt to Income Ratio
* Loan Amount
* Interest Rate
* Loan Term
* Employment Status
* Loan Paid Back

## Analysis Performed

The project includes the following visualizations:

* Age Distribution
* Credit Score Distribution
* Debt to Income Ratio Distribution
* Income vs Loan Amount
* Interest Rate by Employment Status
* Loan Paid Back Count

## Output

The analysis generates visualization charts which are saved as PNG files:

* age_distribution.png
* credit_score_distribution.png
* debt_to_income_distribution.png
* income_vs_loan.png
* interest_by_employment.png
* loan_paid_back_count.png

## How to Run the Project

1. Install required libraries:

pip install -r requirements.txt

2. Run the Python script:

python bank_loan.py

## Project Structure

bank_loan.py – Python script for analysis
project_500.xlsx – Dataset used for analysis
PNG files – Generated charts and visualizations
